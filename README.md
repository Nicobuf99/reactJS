# Ecommerce de mates y productos similares

Este proyecto, diseñado y completado con la libreria react JS, permite de forma dinamica mostrar un e-commerce funcional para seleccionar y comprar productos

## Herramientas utilizadas durante el proyecto

-Lenguaje de Javascript utilizando la libreria de React
-Utilizacion de context para conectar a toda la aplicacion y las funcionalidades del carrito de compras
-Utilizacion del Browser Router para poder conectar a la aplicacion a traves de diferentes Links
-Librerias utilizadas durante el proyecto para la NavBar (Bootstrap) y para la espera de la carga de productos (UIball loaders)
-Utilizacion de firebase para almacenamiento de datos de los productos mas los pedidos de compra de los clientes
